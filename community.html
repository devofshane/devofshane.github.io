Books

Book Name Description
The Pragmatic Programmer: From Journeyman to Master The Pragmatic Programmer doesn't take itself too seriously, giving thought provoking advice in the form of self-contained sections through the use of entertaining anecdotes, thoughtful examplse, and interesting analogies.
Clean Code: A Handbook of Agile Software Craftsmanship Written by the famous "Uncle Bob" Martin, this book provides developers with a handbook, curated by top developers, of the best practices for creating "clean" code. Clean Code provides developers with the knowledge needed to "sniff out" bad code and fix it on the fly.
Code: The Hidden Language of Computer Hardware and Software Using everyday objects and familiar language systems such as Braille and Morse code, author Charles Petzold weaves an illuminating narrative for anyone who's ever wondered about the secret inner life of computers and other smart machines, done in a way that is simple for anyone to follow.
Getting Things Done: The Art of Stress-Free Productivity Not strictly for developers, but even developers need to know how to get things done quickly, efficiently, and without worry. David Allen achieves all of that and then some in this book.
The Passionate Programmer: Creating a Remarkable Career in Software Development This short book is packed full of incredibly useful advice on every single page, making it something that any developer should keep by their side at all times as a handbook. This book will teach you how to go from an average career to an extraordanary career.
The Ten Day MBA This book breaks down what is taught in a typical MBA into ten easily digestable chapters, one for each day it is meant to complete this book. Why read this as a developer? This book will give you the basic (and far beyond basic) knowledge necessary to work well in any business environment.
Cracking the Coding Interview This book is written from the perspective of a software developer who has had an extensive career as a hiring manager, hiring other software developers. This gives her the unique ability to be able to know exactly what a software developer should know for an interview. This book is not strictly for preparing for interviews, but that is it's primary purpose.

Websites

Website Description
Treehouse Treehouse has easy to follow tracks for learning specific languages and technology stacks. It isn't cheap, coming in at roughly $30 a month of the basic plan, but it is well worth the price for the value it can bring to a developer.
edX edX provides courses in many professional areas, including computer science. The bonus is that these courses are all offered and curated by top universities across the world and freely available for anyone to take. You can also get a certificate for the course at a fee.
MIT OpenCourseWare MIT OpenCourseWare is a Website that has been around for over 15 years now, ancient in internet years. MIT offers nearly all of the same courses they offer on campus in a form that any person can take, completely for free. Many of the courses have video lectures, assignments, notes, and more for a top notch level of learning.
Learn Programming Subreddit Reddit has subreddits for nearly everything. If you don't know about Reddit, it is simply a Website that offeres users the ability to make 'subreddits' for whatever topic they desire and create communities around those subreddits. The Learn Programming subreddit is one of the most popular for developers.
CS Career Questions Subreddit The CS career questions subreddit gives a place for people to ask any question they might have about working in the computer science field. This subreddit is another very active one in the community that provides a valuable resource to talk directly with other developers who have jobs in the industry.
Hacker News Hacker News is a great Website to use to stay up-to-date with everything going on in the hacker community. Hacker meaning developers, technology enthusiasts, etc. The Website uses a very simple design making it easy and quick to consume whatever news has been submitted by the community.

This page contains a list of various resources; books, learning Websites, and communities that I take advantage of to help me improve myself as a developer. There are, of course, many many more resources I use on a regular basis, but that would be a list far too large to easily maintain.
